1.1 概率空间

随机现象的严格数学载体

随机试验与样本空间

随机试验是概率论的基本概念，虽然试验的结果事先不能准确预言，但具有如下三个特征：

  • （1）可以在相同的条件下重复进行；
  • （2）每次试验的结果不止一个，但预先知道试验的所有可能的结果；
  • （3）每次试验前不能确定哪个结果会出现。

随机试验的可能结果称为样本点或基本事件，记为 \(\omega\)。
样本点的全体称为样本空间，记为 \(\Omega\)。
样本空间 \(\Omega\) 称为必然事件，空集 \(\varnothing\) 称为不可能事件。\(\Omega\) 的子集 \(A\) 由基本事件组成，通常称为事件。

核心概念：样本点、样本空间与事件

“掷骰子时，‘1、2、3、4、5、6’是一个个可能结果，也就是 样本点 \((\omega)\)； 这些结果合在一起就是 样本空间 \((\Omega)\)； 从里面挑出一部分，比如“偶数点”“大于 4 的点数”，就构成了 事件 \((A,B)\)。”

样本空间 \(\Omega=\{1,2,3,4,5,6\}\)
事件 \(A=\{2,4,6\}\)：偶数点
事件 \(B=\{5,6\}\)：大于 4
样本点 \(\omega_1=1\)
样本点 \(\omega_3=3\)
\(\omega_2 \in A\)
\(\omega_4 \in A\)
\(\omega_5 \in B\)
\(\omega_6 \in A \cap B\)
交集：同时满足

集合的基本运算律

  • (1) 交换律：\(A \cup B = B \cup A,\quad A \cap B = B \cap A\)
  • (2) 结合律：\(A \cup (B \cup C) = (A \cup B) \cup C,\quad A \cap (B \cap C) = (A \cap B) \cap C\)
  • (3) 分配律： \[ A \cup (B \cap C) = (A \cup B) \cap (A \cup C) \] \[ A \cap (B \cup C) = (A \cap B) \cup (A \cap C) \]
  • (4) 对偶原则（De Morgan 律）： \[ \overline{A \cup B} = \bar{A} \cap \bar{B},\qquad \overline{A \cap B} = \bar{A} \cup \bar{B} \] \[ \overline{\bigcup_{i=1}^{\infty} A_i} = \bigcap_{i=1}^{\infty} \overline{A_i},\qquad \overline{\bigcap_{i=1}^{\infty} A_i} = \bigcup_{i=1}^{\infty} \overline{A_i} \]
\[ \text{小结：并、交、补三种运算满足交换、结合、分配与对偶关系} \]

定义 1.1.1：\(\sigma\) 代数与可测空间

设 \(\Omega\) 是一个样本空间，\(\mathscr{F}\) 是 \(\Omega\) 的某些子集组成的集合族。如果满足：

（1）\(\Omega \in \mathscr{F}\) （全集要能讨论）

（2）若 \(A \in \mathscr{F}\)，则 \(A^c = \Omega \setminus A \in \mathscr{F}\) (能讨论一件事，就要能讨论它“不发生”)

（3）若 \(A_n \in \mathscr{F}\ (n=1,2,\dots)\)，则 \(\bigcup_{n=1}^{\infty} A_n \in \mathscr{F}\)(能讨论可数多个事件，就要能讨论“至少一个发生”)

则称 \(\mathscr{F}\) 为 \(\Omega\) 上的一个 \(\sigma\) 代数(“允许讨论、允许赋概率”的那些事件的集合)，\((\Omega, \mathscr{F})\) 称为可测空间(把“结果空间”和“可讨论事件规则”一起打包)。

推论：如果 \(\mathscr{F}\) 是 \(\sigma\) 代数，则必有 \(\varnothing \in \mathscr{F}\)，且对可列交运算 \(\bigcap_{n=1}^{\infty} A_n \in \mathscr{F}\) 封闭。

由某集类 \(\mathscr{C}\) 生成的最小 \(\sigma\) 代数记为 \(\sigma(\mathscr{C})\)。

定义 1.1.2：Borel \(\sigma\) 代数

设 \(\Omega = \mathbb{R}\)。

由所有半无限区间 \((-\infty, x), x \in \mathbb{R}\) 生成的 \(\sigma\) 代数称为 \(\mathbb{R}\) 上的 Borel \(\sigma\) 代数，记为 \(\mathscr{B}(\mathbb{R})\)。

其中的元素称为 Borel 集合。

类似地，可定义 \(\mathbb{R}^n\) 上的 Borel \(\sigma\) 代数 \(\mathscr{B}(\mathbb{R}^n)\)。

定义 1.1.2：Borel \(\sigma\) 代数

在实数轴上，并不是任意一个奇奇怪怪的集合都适合拿来当事件。我们从最基本的区间出发， 通过补集、可列并、可列交不断生成一大类“规矩的集合”，这类集合就叫 Borel 集， 它们组成的集合族叫 Borel \(\sigma\) 代数。

定义 1.1.3：集合序列的极限

设 \(\{A_n, n=1,2,\dots\}\) 为一集合序列。分别称：

\[\limsup_{n \to \infty} A_n = \bigcap_{n=1}^{\infty} \bigcup_{k=n}^{\infty} A_k, \quad \liminf_{n \to \infty} A_n = \bigcup_{n=1}^{\infty} \bigcap_{k=n}^{\infty} A_k\]

上极限 (\(\limsup\))

\[\{w \mid \forall n \in \mathbb{N}, \exists k \ge n, \text{使 } w \in A_k\}\]

即 \(\{w \mid w \text{ 属于无穷多个 } A_n\}\)

下极限 (\(\liminf\))

\[\{w \mid \exists n \in \mathbb{N}, \forall k \ge n, \text{有 } w \in A_k\}\]

即 \(\{w \mid w \text{ 至多不属于有限多个 } A_n\}\)

从而恒有：\(\liminf_{n\to\infty} A_n \subset \limsup_{n\to\infty} A_n\)。若二者相等，则称极限 \(\lim_{n\to\infty} A_n\) 存在。

集合序列极限的例题

例 1.1.6：单调序列

设 \(A_n = [0, 1 + \frac{1}{n}]\)。显然 \(A_n\) 是一个递减集合序列。

\[\limsup_{n\to\infty} A_n = \bigcap_{n=1}^{\infty} \left( \bigcup_{k=n}^{\infty} \left[0, 1+\frac{1}{k}\right] \right) = \bigcap_{n=1}^{\infty} \left[0, 1+\frac{1}{n}\right] = [0,1]\] \[\liminf_{n\to\infty} A_n = \bigcup_{n=1}^{\infty} \left( \bigcap_{k=n}^{\infty} \left[0, 1+\frac{1}{k}\right] \right) = \bigcup_{n=1}^{\infty} [0,1] = [0,1]\]

例 1.1.8：抛硬币的直观理解

  • \(\limsup_{n\to\infty} A_n\) = {有无限多个抛掷结果是“正面”}
  • \(\liminf_{n\to\infty} A_n\) = {除有限多个外，抛掷结果都是“正面”}

定义 1.1.4：概率的公理化定义

设 \((\Omega, \mathscr{F})\) 是可测空间，\(P(\cdot)\) 是定义在 \(\mathscr{F}\) 上的实值函数。如果：

  • （1）规范性： \(P(\Omega) = 1\)
  • （2）非负性： \(\forall A \in \mathscr{F},\ 0 \le P(A) \le 1\)
  • （3）可列可加性： 对两两互不相容事件 \(A_1, A_2, \dots\) （即 \(i \neq j\) 时 \(A_i \cap A_j = \varnothing\)），有：
    \[ P\left(\bigcup_{i=1}^{\infty} A_i\right)=\sum_{i=1}^{\infty}P(A_i) \]

则称 \(P\) 是 \((\Omega, \mathscr{F})\) 上的概率，\((\Omega, \mathscr{F}, P)\) 称为概率空间。

例 1.1.10：验证泊松测度

设某股票一天成交笔数 \(m\)，样本空间 \(\Omega=\{0, 1, 2, \dots\}\)，\(\mathscr{F}\) 为一切子集。定义 \(P(\varnothing)=0\)，并对 \(A \in \mathscr{F}\) 令：

\[P(A) = \sum_{k \in A} \mathrm{e}^{-\lambda} \frac{\lambda^k}{k!}, \quad \lambda>0\]

证明：\(P\) 为可测空间上的概率测度。

(1) 证明 \(P(\Omega)=1\)：

\[P(\Omega) = \sum_{k \in \Omega} \mathrm{e}^{-\lambda} \frac{\lambda^k}{k!} = \sum_{k=0}^{\infty} \mathrm{e}^{-\lambda} \frac{\lambda^k}{k!} = \mathrm{e}^{-\lambda} \mathrm{e}^\lambda = 1\]

例 1.1.10：验证泊松测度

(2) 可列可加性：对于互不相交的集合序列 \(A_n\)

\[P\left(\bigcup_{n=1}^{\infty} A_n\right) = \sum_{k \in \cup_n A_n} \mathrm{e}^{-\lambda} \frac{\lambda^k}{k!} = \sum_{n=1}^{\infty} \sum_{k \in A_n} \mathrm{e}^{-\lambda} \frac{\lambda^k}{k!} = \sum_{n=1}^{\infty} P(A_n)\]

因此 \(P\) 是概率测度。

概率的基本性质

  • (1) \(P(A \cup B) + P(A \cap B) = P(A) + P(B)\)
  • (2) 若 \(A \subset B\)，则 \(P(B-A) = P(B) - P(A)\)（可减性）
  • (3) 若 \(A \subset B\)，则 \(P(A) \le P(B)\)（单调性）
  • (4) 次 \(\sigma\) 可加性： \(P\left(\bigcup_{n \ge 1} A_n\right) \le \sum_{n \ge 1} P(A_n)\)
  • (5) 从下连续： 若 \(A_n \uparrow A\)，则 \(P(A)=\lim_{n\to\infty}P(A_n)\)
  • (6) 从上连续： 若 \(A_n \downarrow A\)，则 \(P(A)=\lim_{n\to\infty}P(A_n)\)
\[\text{小结：可列可加性 } \Longrightarrow \text{ 单调性、次可加性、连续性}\]

1.2 随机变量与分布函数

将抽象事件映射到实数域的桥梁

定义 1.2.1：随机变量

设 \((\Omega, \mathscr{F}, P)\) 是概率空间，\(X\) 是定义在 \(\Omega\) 上取值于 \(\mathbb{R}\) 的函数。

如果对 \(\forall x \in \mathbb{R}\)，有

\[ \{\omega: X(\omega) \le x\} \in \mathscr{F} \]

则称 \(X\) 是 \(\mathscr{F}\) 上的随机变量。

函数

\[ F(x) = P\{\omega: X(\omega) \le x\}, \quad -\infty < x < +\infty \]

称为随机变量 \(X\) 的分布函数。

注：我们要求 \(\{\omega: X(\omega) \le x\} \in \mathscr{F}\) 是为了确保它是一个“可测事件”，这样我们才能用概率测度 \(P\) 去衡量它发生的概率。

例 1.2.1：可测性的直观理解

设 \(\Omega=\{w_1, w_2, w_3\}, \mathscr{F}=\{\varnothing, \Omega, \{w_1, w_2\}, \{w_3\}\}\)。定义 \(X, Y\)：

\[ X = \begin{cases} 1, & \{w_1, w_2\} \\ 2, & \{w_3\} \end{cases}, \quad Y = \begin{cases} 1, & \{w_1\} \\ 2, & \{w_2, w_3\} \end{cases} \]

检查原像是否在 \(\mathscr{F}\) 中：

\[ \{X=1\} = \{w_1, w_2\} \in \mathscr{F}, \quad \{X=2\} = \{w_3\} \in \mathscr{F} \] \[ \{Y=1\} = \{w_1\} \notin \mathscr{F}, \quad \{Y=2\} = \{w_2, w_3\} \notin \mathscr{F} \]

结论：\(X\) 是关于 \(\mathscr{F}\) 的随机变量，而 \(Y\) 不是。

如果定义新的代数 \(\hat{\mathscr{F}}\) 包含 \(\{w_1\}, \{w_2\}, \{w_3\}\) 等所有子集，那么 \(X, Y\) 就都是关于 \(\hat{\mathscr{F}}\) 的随机变量了。可见随机变量是相对于特定的 \(\sigma\) 代数而言的。

随机变量的运算封闭性

定理 1.2.1 可测性等价条件： \(\{X \ge x\}, \{X \gt x\}, \{X \lt x\}\) 都属于 \(\mathscr{F}\)。

(1)：若 \(X, Y\) 是随机变量，则 \(\{X\lt Y\}\in\mathscr{F}\)

(2)：\(X+Y\)、\(X^2\) 与 \(XY\) 都是随机变量

(3)：\(\sup X_n\)、\(\limsup X_n\) 仍是随机变量

结论：随机变量在比较、四则运算、上确界和极限运算下都保持可测性。

这也是后续研究随机过程、极限定理与鞅时能够自由进行代数和极限操作的基础。

常见分布 (1)：离散型分布

1. 退化分布 (单点)

\(P\{X=c\} = 1\)

2. Bernoulli 分布

\(P\{X=k\} = p^k(1-p)^{1-k}, \quad k=0,1\)

3. 二项分布 \(B(n,p)\)

\[ P\{X=k\} = \binom{n}{k} p^k(1-p)^{n-k} \]

4. Poisson 分布 \(P(\lambda)\)

\[ P\{X=k\} = \mathrm{e}^{-\lambda} \frac{\lambda^k}{k!} \]

常见分布 (1-续)：离散型分布

5. 几何分布

\[ P\{X=k\} = p(1-p)^{k-1}, \quad k \ge 1 \]

6. 离散均匀分布

\[ P\{X=k\} = \frac{1}{n}, \quad 1 \le k \le n \]

7. 负二项分布

在 Bernoulli 试验序列中，事件 A 第 r 次出现 的试验次数 \(k\)：

\[P\{X=k\} = \binom{k+r-1}{k} p^r(1-p)^k, \quad k=0,1,2, \dots\]

注：当某类数据集观测到的方差大于均值时，负二项分布比泊松分布（均值=方差）更合适。

常见分布 (2)：连续型分布（上）

8. 均匀分布

\[ f(x)=(b-a)^{-1},\quad x\in[a,b] \]

9. 正态 / Gauss 分布 \(N(\mu,\sigma^2)\)

\[ f(x)=\frac{1}{\sigma\sqrt{2\pi}} \exp\left\{-\frac{(x-\mu)^2}{2\sigma^2}\right\} \]

10. \(d\) 维正态分布 \(N_d(\boldsymbol{\mu},\boldsymbol{\Sigma})\)

\[ f(\boldsymbol{x}) =(2\pi)^{-d/2}|\boldsymbol{\Sigma}|^{-1/2} \exp\left\{ -\frac{1}{2} (\boldsymbol{x}-\boldsymbol{\mu})' \boldsymbol{\Sigma}^{-1} (\boldsymbol{x}-\boldsymbol{\mu}) \right\} \]

常见分布 (2)：连续型分布（下）

11. \(\Gamma\) 分布

\[ f(x)=\frac{\lambda^s}{\Gamma(s)} x^{\,s-1}\mathrm{e}^{-\lambda x} \]

12. 指数分布

当 \(\Gamma\) 分布中取 \(s=1\) 时，就得到指数分布：

\[ f(x)=\lambda \mathrm{e}^{-\lambda x} \]

13. \(\chi^2\) 分布

在 \(\Gamma\) 分布中取 \(s=n/2,\ \lambda=1/2\)，可得 \(\chi^2\) 分布：

\[ f(x)=\frac{1}{2^{n/2}\Gamma(n/2)} x^{n/2-1}\mathrm{e}^{-x/2} \]

1.3 数字特征、矩母函数与特征函数

分布的粗略轮廓与唯一“指纹”

核心工具：黎曼-斯蒂尔杰斯积分

“在概率论中，我们不再只对空间 \(dx\) 均匀积分，而是对 ‘概率的累积量 \(dF(x)\)’ 进行积分。黎曼-斯蒂尔杰斯积分 \(\int f(x)\,dg(x)\) 允许‘底边宽度’由另一个函数 \(g(x)\) 决定， 它将离散求和与连续积分统一到同一个公式下。”

1. 几何意义：非均匀加权
\(\int f(x)\,dg(x)\approx \sum f(\xi_i)\,[g(x_i)-g(x_{i-1})]\)
直观理解：高度由 \(f(x)\) 决定，但底边不再是 \(\Delta x\)，而是 \(\Delta g(x)\)。若 \(g(x)\) 变化更快，该段权重更大。
2. 概率论应用：统一离散与连续
如果 \(F(x)\) 是平滑曲线
\(dF(x)=f(x)\,dx\)
退化为普通积分
\(E[X]=\int x\,dF(x)\)
如果 \(F(x)\) 是阶跃函数
\(\int x\,dF(x)=x_1p_1+x_2p_2\)
跳点处变成求和
核心结论：R-S 积分把离散求和与连续积分统一写成 \(\int f(x)\,dF(x)\)，这正是概率论里期望表达式统一的关键。

1.3.1 数学期望与方差 (R-S 积分)

利用 Riemann-Stieltjes 积分，可以对离散和连续的期望给出统一表达式：

\[E(X) = \int_{-\infty}^{+\infty} x \mathrm{d} F(x)\]

方差 (Variance)

\[\operatorname{Var}(X) = \int_{-\infty}^{+\infty} [x - E(X)]^2 \mathrm{d} F(x)\]

即二阶中心矩。

协方差 (Covariance)

\[\operatorname{Cov}(X, Y) = E[(X-EX)(Y-EY)]\]
\[= E(XY) - E(X)E(Y)\]

\(E(X^k)\) 称为 \(k\) 阶原点矩；\(E[X - E(X)]^k\) 称为 \(k\) 阶中心矩。

例 1.3.1：对数正态分布的期望与方差

假设 \(X \sim N(\mu, \sigma^2)\)，求 \(Y = \mathrm{e}^X\) 的期望和方差。

求期望 \(E(Y)\)：

\[ \begin{aligned} E(Y) &= E(\mathrm{e}^X) = \frac{1}{\sqrt{2\pi}\sigma} \int_{-\infty}^{+\infty} \mathrm{e}^x \cdot \mathrm{e}^{-\frac{(x-\mu)^2}{2\sigma^2}} \mathrm{d}x \\ &= \frac{1}{\sqrt{2\pi}\sigma} \int_{-\infty}^{+\infty} \mathrm{e}^{\mu + \frac{\sigma^2}{2}} \cdot \mathrm{e}^{-\frac{(x - (\mu+\sigma^2))^2}{2\sigma^2}} \mathrm{d}x \\ &= \mathrm{e}^{\mu + \frac{\sigma^2}{2}} \end{aligned} \]

求方差 \(\operatorname{Var}(Y)\)：

因 \(Y^2 = \mathrm{e}^{2X}, X \sim N(\mu, \sigma^2)\)，故有 \(2X \sim N(2\mu, 4\sigma^2)\)。代入上述公式：

\[E(Y^2) = \mathrm{e}^{2\mu + 2\sigma^2}\] \[D(Y) = E(Y^2) - [E(Y)]^2 = \mathrm{e}^{2\mu + 2\sigma^2} - \mathrm{e}^{2\mu + \sigma^2}\]

例 1.3.2：匹配问题 (帽子问题)

\(n\) 个人将帽子打乱后每人随机取一顶。\(X\) 表示取到自己帽子的人数，求 \(X\) 的期望和方差。

解：引入示性随机变量 \(X_i = 1\) (若第 \(i\) 个人取到了自己的帽子)，否则为 0。则 \(X = \sum_{i=1}^n X_i\)。

\[P\{X_i=1\} = \frac{1}{n}, \quad E(X_i) = \frac{1}{n}, \quad \operatorname{Var}(X_i) = \frac{1}{n} - \frac{1}{n^2}\]
\[E(X) = E\left[\sum_{i=1}^n X_i\right] = n \times \frac{1}{n} = 1\]

求方差需计算协方差：

\[\operatorname{Var}(X) = \sum_{i=1}^n \operatorname{Var}(X_i) + 2 \sum_{1 \le i < j \le n} \operatorname{Cov}(X_i, X_j)\]

例 1.3.3：正态分布的 k 阶原点矩

若 \(X \sim N(0, \sigma^2)\)，求 \(E(X^k)\)。

\[E(X^k) = \int_{-\infty}^{+\infty} x^k \frac{1}{\sqrt{2\pi}\sigma} \mathrm{e}^{-\frac{x^2}{2\sigma^2}} \mathrm{d}x\]

(1) 当 \(k\) 为奇数时，被积函数为奇函数，积分区间对称，故 \(E(X^k)=0\)。

(2) 当 \(k\) 为偶数时。令 \(t = \frac{x^2}{2\sigma^2}\)，即 \(x = \sqrt{2} \sigma t^{1/2}, \mathrm{d}x = \frac{\sqrt{2}\sigma}{2} t^{-1/2} \mathrm{d}t\)：

\[ \begin{aligned} E(X^k) &= 2 \int_0^{+\infty} \frac{1}{\sqrt{2\pi}\sigma} (\sqrt{2}\sigma t^{1/2})^k \mathrm{e}^{-t} \frac{\sqrt{2}\sigma}{2} t^{-1/2} \mathrm{d}t \\ &= \frac{2^{k/2}\sigma^k}{\sqrt{\pi}} \int_0^{+\infty} t^{\frac{k+1}{2}-1} \mathrm{e}^{-t} \mathrm{d}t = \frac{2^{k/2}\sigma^k}{\sqrt{\pi}} \Gamma\left(\frac{k+1}{2}\right) \\ &= \frac{2^{k/2}\sigma^k}{\sqrt{\pi}} \times \frac{k-1}{2} \times \frac{k-3}{2} \dots \times \frac{1}{2} \times \Gamma\left(\frac{1}{2}\right) \\ &= \sigma^k(k-1)(k-3)\dots 3 \times 1 \quad \text{(由于 } \Gamma(1/2) = \sqrt{\pi}) \end{aligned} \]

1.3.3 矩母函数 (MGF)

定义 1.3.2：若随机变量 \(X\) 的分布函数为 \(F(x)\)，则称

\[\phi(t) = E(\mathrm{e}^{tX}) = \int_{-\infty}^{+\infty} \mathrm{e}^{tx} \mathrm{d} F(x)\]

为 \(X\) 的矩母函数。

性质：通过对 \(\phi(t)\) 逐次求导并计算在 \(t=0\) 点的值就能得到 \(X\) 的各阶矩：

\[\phi'(t) = E(X \mathrm{e}^{tX}) \quad \Rightarrow \quad \phi'(0) = E(X)\] \[\phi''(t) = E(X^2 \mathrm{e}^{tX}) \quad \Rightarrow \quad \phi''(0) = E(X^2)\] \[\phi^{(n)}(t) = E(X^n \mathrm{e}^{tX}) \quad \Rightarrow \quad \phi^{(n)}(0) = E(X^n)\]

当矩母函数存在时，它唯一决定分布。但有时矩母函数不一定存在（积分发散），此时特征函数更方便。

1.3.4 特征函数 (CF)

定义 1.3.3：

\[\psi(t) = E(\mathrm{e}^{\mathrm{i}tX}) = \int_{-\infty}^{+\infty} \mathrm{e}^{\mathrm{i}tx} \mathrm{d} F(x)\]

为 \(X\) 的特征函数。如果存在密度 \(f(x)\)，则 \(\psi(t)\) 就是 \(f(x)\) 的 Fourier 变换。

1.3.4 特征函数 (CF)

因为 \(|\mathrm{e}^{\mathrm{i}tx}| = 1\)，所以特征函数对一切实数 \(t\) 都有定义。 常用性质：

  • (1) 有界性：\(|\psi(t)| \le 1 = \psi(0)\)
  • (2) 共轭对称性：\(\psi(-t) = \overline{\psi(t)}\) (共轭:\(z=a+bi,\overline{z}=a-bi\))
  • (4) 线性变换：设 \(Y = aX+b\)，则 \(\psi_Y(t) = \mathrm{e}^{\mathrm{i}bt} \psi_X(at)\)
  • (5) 两个相互独立的随机变量之和的特征函数等于它们的特征函数之积。
  • (5) 矩的关系：若 \(n\) 阶矩存在，则 \(\psi^{(k)}(0) = \mathrm{i}^k E(X^k)\)
课堂练习

特征函数计算实例 (1/3)

例 1.3.4：单点分布

\(P\{X=c\}=1\) 的特征函数：

\[\psi(t) = E(\mathrm{e}^{\mathrm{i}tc}) = \mathrm{e}^{\mathrm{i}tc}\]

例 1.3.5：两点分布 (Bernoulli)

\(P\{X=0\}=p, P\{X=1\}=1-p\)：

\[\psi(t) = p \mathrm{e}^{\mathrm{i}t\cdot 0} + (1-p)\mathrm{e}^{\mathrm{i}t\cdot 1} = p + \mathrm{e}^{\mathrm{i}t}(1-p)\]

特征函数计算实例 (2/3)

例 1.3.6：指数分布的特征函数

密度 \(f(x) = \lambda \mathrm{e}^{-\lambda x} (x \ge 0)\)。利用欧拉公式展开：

\[ \begin{aligned} \psi(t) &= \int_0^{+\infty} \lambda \mathrm{e}^{\mathrm{i}tx} \mathrm{e}^{-\lambda x} \mathrm{d}x \\ &= \int_0^{+\infty} \lambda \mathrm{e}^{-\lambda x} \cos(tx) \mathrm{d}x + \mathrm{i}\lambda \int_0^{+\infty} \mathrm{e}^{-\lambda x} \sin(tx) \mathrm{d}x \end{aligned} \]

回顾数学分析中的分部积分（以第一项为例）：

\[ \begin{aligned} \int_0^{+\infty} \mathrm{e}^{-\lambda x} \cos x \mathrm{d}x &= \mathrm{e}^{-\lambda x} \sin x \Big|_0^{+\infty} + \lambda \int_0^{+\infty} \mathrm{e}^{-\lambda x} \sin x \mathrm{d}x \\ &= \lambda \left[ -\mathrm{e}^{-\lambda x} \cos x \Big|_0^{+\infty} - \lambda \int_0^{+\infty} \mathrm{e}^{-\lambda x} \cos x \mathrm{d}x \right] \\ &= \lambda - \lambda^2 \int_0^{+\infty} \mathrm{e}^{-\lambda x} \cos x \mathrm{d}x \quad \Rightarrow \quad \int = \frac{\lambda}{1+\lambda^2} \end{aligned} \]

代回原式化简得：\(\psi(t) = \lambda \frac{\lambda}{\lambda^2+t^2} + \mathrm{i}\lambda \frac{t}{\lambda^2+t^2} = (1 - \frac{\mathrm{i}t}{\lambda})^{-1}\)

特征函数计算实例 (3/3)

例 1.3.7 & 1.3.8：正态分布的特征函数

对于 \(X \sim N(0,1)\)，由偶函数对称性，\(\sin\) 项积分抵消：

\[\psi_X(t) = \frac{2}{\sqrt{2\pi}} \int_0^{+\infty} \cos(tx) \mathrm{e}^{-x^2/2} \mathrm{d}x\]

两边对 \(t\) 求导并利用分部积分构造常微分方程：

\[ \psi_X'(t) = \frac{2}{\sqrt{2\pi}} \int_0^{+\infty} \sin(tx) \mathrm{d}\mathrm{e}^{-x^2/2} = -t \psi_X(t) \]

解得初值问题 \(\psi_X(0)=1 \Rightarrow \psi_X(t) = \mathrm{e}^{-t^2/2}\)。

对于 \(Y \sim N(\mu, \sigma^2)\)，有 \(Y = \sigma X + \mu\)，利用线性性质：

\[\psi_Y(t) = \mathrm{e}^{\mathrm{i}t\mu} \psi_X(\sigma t) = \mathrm{e}^{\mathrm{i}t\mu - \frac{1}{2}\sigma^2 t^2}\]

例 1.3.9：Poisson 的特征函数

设 \(X\sim P(\lambda)\)。计算特征函数：

\[ \begin{aligned} \psi_X(t) &= \sum_{x=0}^{\infty}\mathrm{e}^{\mathrm{i}tx}\frac{\lambda^x}{x!}\mathrm{e}^{-\lambda} = \mathrm{e}^{-\lambda}\sum_{x=0}^{\infty}\frac{(\lambda \mathrm{e}^{\mathrm{i}t})^x}{x!}\\ &= \mathrm{e}^{-\lambda}\mathrm{e}^{\lambda \mathrm{e}^{\mathrm{i}t}} = \mathrm{e}^{\lambda(\mathrm{e}^{\mathrm{i}t}-1)} \end{aligned} \]

因此，Poisson 分布的特征函数为

\[\psi_X(t)=\mathrm{e}^{\lambda(\mathrm{e}^{\mathrm{i}t}-1)}\]

例 1.3.9：Poisson 的矩与方差

由 \(\psi^{(k)}(0)=\mathrm{i}^k E(X^k)\) 求矩：

\[ E(X)=-\mathrm{i}\psi'(0)=\lambda \] \[ E(X^2)=-\psi''(0)=\lambda+\lambda^2 \]
\[ \operatorname{Var}(X)=E(X^2)-[E(X)]^2=\lambda \]

结论：Poisson 分布的均值与方差都等于 \(\lambda\)。

1.4 收敛性

随机变量序列的四种极限界限

定义 1.4.1：四大收敛性

(1) 几乎必然收敛 (a.s.)

\[P\{\omega: \lim_{n\to\infty} X_n(\omega) = X(\omega)\} = 1\]

(最强，样本轨道级别收敛)

(2) 依概率收敛 (P)

\[\lim_{n\to\infty} P\{|X_n - X| \ge \varepsilon\} = 0\]

(大数定律基础)

(3) L^p 平均收敛

\[\lim_{n\to\infty} E(|X_n - X|^p) = 0\]

(p=2为均方收敛)

(4) 依分布收敛 (弱收敛 / d)

\[\lim_{n\to\infty} F_n(x) = F(x)\]

(最弱，中心极限定理基础)

收敛性的关系与反例 (例 1.4.1)

几乎必然收敛 \(\Rightarrow\) 依概率收敛 \(\Rightarrow\) 依分布收敛

L^p 收敛 \(\Rightarrow\) 依概率收敛 \(\Rightarrow\) 依分布收敛

1.5 独立性与条件期望

随机过程建模的最核心工具：预测未来

1.5.1 独立性

相互独立：事件组中任意有限子集的交概率等于概率的乘积。

定理 1.5.1：独立变量乘积的期望=期望的乘积；独立变量和的方差=方差的和。

1.5.2 独立随机变量和的分布 (卷积)

设 \(X_1, X_2\) 独立，分布函数分别为 \(F_1, F_2\)。令 \(X = X_1 + X_2\)，则：

\[F_X(x) = \int_{-\infty}^{+\infty} F_2(x-t) \mathrm{d} F_1(t) \triangleq (F_1 * F_2)(x)\]

这称为 卷积 (Convolution)。

  • 交换律：\(F_1 * F_2 = F_2 * F_1\)
  • 结合律：\((F * G) * H = F * (G * H)\)
  • 分配律：\(F * (G+H) = F * G + F * H\)
  • 同分布 n 重卷积：\(F_n(x) = F * F_{n-1}(x)\)

1.5.3 条件期望 (最优预测)

\(E(X \mid Y)\) 表示已知 Y 的信息后，对 X 的预测。

离散情形

\[E(X \mid Y=y) = \sum_x x P\{X=x \mid Y=y\}\]

连续情形

\[E(X \mid Y=y) = \int x f(x \mid y) \mathrm{d} x\]

全期望公式 (极其重要)

\[E(X) = E[E(X \mid Y)] = \int E(X \mid Y=y) \mathrm{d} F_Y(y)\]

注：\(E(X \mid Y=y)\) 是实数函数；\(E(X \mid Y)\) 是一个随机变量。

条件期望的应用例题 (1)

例 1.5.1：求连续条件期望

联合密度 \(f(x,y) = \frac{1}{2}\mathrm{e}^{-y} \ (y>|x|)\)。求 \(E(Y|X=x)\)。

边缘密度 \(f_X(x) = \int_{|x|}^\infty \frac{1}{2}\mathrm{e}^{-y} \mathrm{d}y = \frac{1}{2}\mathrm{e}^{-|x|}\)。
条件密度 \(f(y|x) = f(x,y)/f_X(x) = \mathrm{e}^{-y+|x|}\)。
积分 \(E(Y|X=x) = \int_{|x|}^\infty y \mathrm{e}^{-y+|x|} \mathrm{d}y = 1 + |x|\)。

例 1.5.2：随机个随机变量之和 (Wald等式)

\(N\) 与独立同分布序列 \(X_i\) 独立，求 \(Y = \sum_{i=1}^N X_i\) 的均值和方差。

先求条件 MGF：\(E[\mathrm{e}^{t \sum X_i} \mid N] = [\phi_X(t)]^N\)。
全期望求 MGF：\(\phi_Y(t) = E\{[\phi_X(t)]^N\}\)。求一阶、二阶导后在 \(t=0\) 取值：
\(E(Y) = E(N)E(X)\)
\(\operatorname{Var}(Y) = E(N)\operatorname{Var}(X) + [E(X)]^2\operatorname{Var}(N)\)

条件期望的应用例题 (2)

例 1.5.3：矿工迷路问题

三个门概率均等：1门(安全, 2小时)，2门(绕回, 3小时)，3门(绕回, 5小时)。求到达安全时间 \(X\) 的期望与矩母函数。

令 \(Y\) 为首次选择的门。利用全期望 \(E(X) = E[E(X|Y)]\)：

\[E(X) = \frac{1}{3}[2 + (3+E(X)) + (5+E(X))] \implies E(X) = 10\]

利用全期望求矩母函数：

\[ \begin{aligned} E(\mathrm{e}^{tX}) &= \frac{1}{3}[E(\mathrm{e}^{tX}|Y=1) + E(\mathrm{e}^{tX}|Y=2) + E(\mathrm{e}^{tX}|Y=3)] \\ &= \frac{1}{3}[\mathrm{e}^{2t} + E(\mathrm{e}^{t(3+X')}) + E(\mathrm{e}^{t(5+X')})] \\ &= \frac{1}{3}[\mathrm{e}^{2t} + \mathrm{e}^{3t}E(\mathrm{e}^{tX}) + \mathrm{e}^{5t}E(\mathrm{e}^{tX})] \implies E(\mathrm{e}^{tX}) = \frac{\mathrm{e}^{2t}}{3-\mathrm{e}^{3t}-\mathrm{e}^{5t}} \end{aligned} \]

定义 1.5.3：一般条件期望与定理\(^{**}\)

基于 \(\sigma\) 代数的严谨定义：设子代数 \(\mathcal{G} \subset \mathcal{F}\)，随机变量 \(X^* = E(X \mid \mathcal{G})\) 必须是 \(\mathcal{G}\) 可测的，且在 \(\mathcal{G}\) 中的任何集合 \(B\) 上积分与 \(X\) 等价：\(\int_B X^* \mathrm{d}P = \int_B X \mathrm{d}P\)。

定理 1.5.5 核心性质

  • (1) 平滑法则： \(E[E(X \mid \mathcal{G})] = E(X)\)
  • (2) 已知即常量： 若 \(X\) 是 \(\mathcal{G}\) 可测的，则 \(E(X \mid \mathcal{G}) = X\) a.s.
  • (6) 线性： \(E(aX+bY \mid \mathcal{G}) = aE(X \mid \mathcal{G}) + bE(Y \mid \mathcal{G})\)
  • (9) 提出来： 若 \(Y\) 是 \(\mathcal{G}\) 可测，\(E(XY \mid \mathcal{G}) = Y E(X \mid \mathcal{G})\) a.s.
  • (10) 独立无关： 若 \(X\) 与 \(\mathcal{G}\) 独立，\(E(X \mid \mathcal{G}) = E(X)\) a.s.
  • (11) 塔律 (Tower)： 若 \(\mathcal{G}_1 \subset \mathcal{G}_2\)，则 \(E[E(X \mid \mathcal{G}_2) \mid \mathcal{G}_1] = E(X \mid \mathcal{G}_1)\) a.s.
