第一节 一元线性回归预测法

最基础的模型：两个变量间的直线拟合

导入实例：为什么叫“回归”？高尔顿的身高之谜

优生学实验的意外发现

1889年，统计学家弗朗西斯·高尔顿在研究人类身高遗传时发现：
高个子父母的孩子通常也较高，但往往没有父母那么高；矮个子父母的孩子也较矮，但往往比父母高。
人类的身高在代代相传中，具有一种“向全人类平均值回归 (Regression)”的强烈趋势。

本节核心：

我们将学习如何用严格的数学方法（最小二乘法），去精确计算出这根代表客观规律的拟合直线 $y_i = b_0 + b_1x_i + u_i$。

一、建立一元线性回归模型

当成对的两个变量数据分布大体上呈直线趋势时，我们建立一元线性回归模型：

\[ y_i = b_0 + b_1 x_i + u_i \quad (3-1) \]
  • $b_0, b_1$： 未知参数（需估计）。
  • $u_i$： 剩余残差项（随机扰动项），包含对 $y_i$ 有影响的所有其他偶然因素。

使用该模型的重要假定（$u_i$ 的 5 个特性）：

  • （1）$u_i$ 是一个随机变量；
  • （2）平均值为零，即 $E(u_i) = 0$；
  • （3）方差为常量（同方差性），即 $D(u_i) = \sigma_u^2$；
  • （4）各个 $u_i$ 间相互独立（无自相关）；
  • （5）$u_i$ 与自变量 $x_i$ 无关。

二、估计参数 (普通最小二乘法 OLS)

估计出 $b_0, b_1$ 后，建立预测式： $\hat{y}_i = b_0 + b_1 x_i$ 。好的估计量应满足一致性、无偏性、有效性。

最小二乘法原理： 使实际值 $y_i$ 与估计值 $\hat{y}_i$ 之间的残差平方和达到最小：

\[ \sum_{i=1}^n u_i^2 = \sum_{i=1}^n (y_i - \hat{y}_i)^2 = \sum_{i=1}^n (y_i - b_0 - b_1 x_i)^2 \rightarrow \min \]

求导并令为零，解得：

\[ b_1 = \frac{\sum(x-\bar{x})(y-\bar{y})}{\sum(x-\bar{x})^2} \] \[ b_0 = \bar{y} - b_1 \bar{x} \]

便于计算的 $b_1$ 等价表达式：

\[ b_1 = \frac{\sum x y - \frac{1}{n}(\sum x)(\sum y)}{\sum x^2 - \frac{1}{n}(\sum x)^2} \] \[ b_1 = \frac{n \sum x y - (\sum x)(\sum y)}{n \sum x^2 - (\sum x)^2} \]

\[ b_1 = \frac{\sum(x-\bar{x})(y-\bar{y})}{\sum(x-\bar{x})^2} = \frac{9855}{1012} = 9.74 \] \[ b_0 = \bar{y} - b_1 \bar{x} = 380 - 9.74 \times 27 = 117 \]

三、进行统计检验 (1/4)

（一）标准误差 (Standard Error)

度量实际值与估计值间的平均平方误差：

\[ SE = \sqrt{\frac{\sum(y-\hat{y})^2}{n-2}} = \sqrt{\frac{33897}{10-2}} = 65 \text{ (箱)} \]

（二）可决系数 $R^2$ (R-squared)

总离差($\mathrm{SST}$) = 回归离差($\mathrm{SSE}$) + 剩余离差($\mathrm{SSR}$)
衡量自变量对因变量变动解释程度的百分比：

\[ R^2 = \frac{\text{SSE}}{\text{SST}} = 1 - \frac{\text{SSR}}{\text{SST}} = 1 - \frac{\sum(y-\hat{y})^2}{\sum(y-\bar{y})^2} = 1 - \frac{33897}{129950} = 0.74 \]

结论：气温因素变化占饮料销售量变动的 74%。

三、进行统计检验 (2/4)

（三）相关系数 $r$

度量线性拟合优度。$r$ 有正负，相关系数的平方即为 $R^2$：

\[ r = \frac{\sum(x-\bar{x})(y-\bar{y})}{\sqrt{\sum(x-\bar{x})^2} \sqrt{\sum(y-\bar{y})^2}} = \frac{9855}{\sqrt{1012} \times \sqrt{129950}} = 0.86 \]

（四）回归系数显著性检验 (t 检验)

检验自变量 $x$ 是否对 $y$ 有显著影响 ($H_0: \beta_1=0$)：

\[ S_b = \frac{SE}{\sqrt{\sum(x-\bar{x})^2}} = \frac{65}{\sqrt{1012}} \doteq 2.05 \quad \Rightarrow \quad t_b = \frac{b_1}{S_b} = \frac{9.74}{2.05} = 4.75 \]

查表 $\alpha=0.05$，自由度 $n-2=8$，得 $t_{0.05}(8) = 2.306$。
由于 $|t_b| = 4.75 > 2.306$，拒绝 $H_0$，说明回归系数显著。

三、进行统计检验 (3/4)

（五）F 检验 (方差分析)

检验模型总体是否显著。检验统计量 $F$ 的计算公式：

\[ F = \frac{\sum(\hat{y}-\bar{y})^2 / 1}{\sum(y-\hat{y})^2 / (n-2)} \]

将例 3-1 中的数据代入：

\[ F = \frac{96053 / 1}{33897 / 8} = 22.67 \]

取显著性水平 $\alpha=0.05$ ，查表得 $F_{0.05}(1,8)=5.32$ 。
因 $F > 5.32$，方程通过 $F$ 检验，表明回归模型整体显著，可用于预测。

三、进行统计检验 (4/4)：德宾-沃森 (D-W) 检验详述

回归模型的剩余项 $u_i$ 之间应相互独立。若存在自相关，预测将失真。德宾-沃森 (D-W) 统计量公式为：

\[ D-W = \frac{\sum_{i=2}^n (u_i - u_{i-1})^2}{\sum_{i=1}^n u_i^2}, \quad \text{其中 } u_i = y_i - \hat{y}_i \]

D-W 检验法则 (取值范围 0 ~ 4)

比较计算值与查表得到的临界值 $d_L$ 和 $d_U$：

  • 当 $D-W \le 2$ 时：
    • $D-W < d_L$：存在正自相关
    • $d_L < D-W < d_U$：不能确定
    • $D-W > d_U$：无自相关
  • 当 $D-W > 2$ 时：
    • $4 - D-W < d_L$ (即 $D-W > 4 - d_L$)：存在负自相关
    • $4 - D-W > d_U$ (即 $D-W < 4 - d_U$)：无自相关
    • 不能确定：其他情况

【图 3-1】D-W 检验法则示意图

经验表明，$D-W = 2$ 时为最好。在 $1.5 \sim 2.5$ 之间表示无显著自相关。

四、进行预测：点估计与区间估计图解

点估计

代入 $x_0=35$，得唯一预测值： $\hat{y} = 117 + 9.74 \times 35 = 458$ (箱)

区间估计 (喇叭口效应)

小样本下使用 $t$ 分布。随着预测点 $x_0$ 远离均值 $\bar{x}$，误差项 $\frac{(x_0-\bar{x})^2}{\sum(x_i-\bar{x})^2}$ 增大，区间呈“喇叭口”状向外发散：

\[ \text{置信区间} = \hat{y} \pm t_{\alpha/2} SE \sqrt{\frac{1}{n} + \frac{(x_0-\bar{x})^2}{\sum(x_i-\bar{x})^2}} \] \[ \text{预测区间} = \hat{y} \pm t_{\alpha/2} SE \sqrt{1 + \frac{1}{n} + \frac{(x_0-\bar{x})^2}{\sum(x_i-\bar{x})^2}} \]

例题代入($\alpha=0.1$):
90%置信区间 (估计 $\mu_y$)： $458 \pm 48$，即 $[410, 506]$ 箱。
90%预测区间 (估计个别值 $y$)： $458 \pm 130$，即 $[328, 588]$ 箱。

第二节 多元线性回归预测法

引入多个影响因素，提高模型预测精度

导入实例：从“二维直线”到“多维超平面”

一元回归的尴尬：房价预测

假设你要预测二手房价格 $y$。如果只使用“房屋面积 $x_1$”，你会发现现实中同为 100平米 的房子，价格却相差数百万！单维度必然导致巨大误差。

升维打击：引入更多信息

要提高精度，必须引入：
$x_2$：距最近地铁站距离
$x_3$：所属学区质量
回归预测从“二维平面上找一根直线”，变成了“在多维空间中寻找一个最优超平面”。

一、多元模型的建立与参数估计

客观事物往往受多种因素影响。以两个自变量为例（引入“比赛比分差 $z$”）：

\[ \hat{y} = b_0 + b_1 x + b_2 z \]

正规方程组（最小二乘法）求解：

\[ \begin{aligned} & \sum y = n b_0 + b_1 \sum x + b_2 \sum z \\ & \sum x y = b_0 \sum x + b_1 \sum x^2 + b_2 \sum x z \\ & \sum z y = b_0 \sum z + b_1 \sum x z + b_2 \sum z^2 \end{aligned} \]

【例 3-2】代入数据解得： $b_0=39.1953, b_1=9.0641, b_2=9.6074$
若气温 $35^\circ\mathrm{C}$，比分差 8，预测饮料销量： $\hat{y} = 39.19 + 9.06\times35 + 9.61\times8 = 433 \text{ (箱)}$

二、多元回归的统计检验

多元回归的 t检验、F检验 核心逻辑与一元相同。但需特别注意以下三项核心检验：

1. 修正可决系数 $R_a^2$

为消除变量增加导致 $R^2$ 虚高，使用修正公式：

\[ R_a^2 = 1 - \frac{\sum(y-\hat{y})^2 / (n-k-1)}{\sum(y-\bar{y})^2 / (n-1)} \]

本例 $R_a^2 = 0.935$，比一元的 0.74 显著提高。

2. 多重共线性检验

自变量间若高度相关会导致模型失效。查相关矩阵：

$r_{xz}=0.13$ 较小，无严重共线性。若极高，需剔除其中一个变量。

3. 自相关检验 (D-W 检验)

检验残差 $u_i$ 序列是否独立：

\[ D-W = \frac{\sum_{i=2}^n (u_i - u_{i-1})^2}{\sum_{i=1}^n u_i^2} \]

对于多元回归（例3-2），计算得 $D-W = 2.19$。
在 $1.5 \sim 2.5$ 范围内，表明无显著自相关。若发现自相关，通常通过对原始数据进行一阶差分来消除。

案例：共线性与使用边界（6/6）

多重共线性直观解释

如果“气温”和“冰块用量”同时进模型，它们可能高度相关。模型会疑惑：销量增加到底是气温的功劳，还是冰块的功劳？这就像小组作业里两个人都说自己贡献最大，最后老师也不知道该给谁加分。

本例处理

  • 选择变量时尽量避免含义重复。
  • 检查自变量之间的相关系数。
  • 若两个自变量高度相关，可剔除其中一个，或用主成分等方法降维。
流程总结：明确问题 $\rightarrow$ 收集数据 $\rightarrow$ 建立模型 $\rightarrow$ OLS估计 $\rightarrow$ 预测 $\rightarrow$ F检验与t检验 $\rightarrow$ 误差评价 $\rightarrow$ 检查共线性与业务逻辑 $\rightarrow$ 给出决策建议。

理解：多元回归不是“变量越多越高级”，而是“重要变量一个别漏，重复变量一个别塞”。不然模型会变成统计版杂货铺，什么都摆上去，最后谁也卖不动。

第三节 非线性回归预测法

通过变量代换，化曲为直

导入实例：直线无法描绘的真实世界

爆款短视频的传播曲线

一条爆款短视频发布后，前三天的播放量可能是：1千、1万、10万。呈现出爆发式的指数级增长。

但到了第十天，由于潜在用户基数见顶，播放量会逐渐停滞在 100万 左右，最终呈现出一条 S型 (Logistic) 饱和曲线。

如果你坚持用一条“直线”去拟合这组数据，无论怎么画，误差都会错得离谱！

数学家的魔法：化曲为直

面对非线性的世界，难道我们熟练掌握的线性回归工具就彻底失效了吗？

统计学家提供了一个极其巧妙的破局思路：变量代换。

只要我们给弯曲的数据戴上一副特殊的“数学眼镜”（比如取对数、求倒数），它在全新的坐标系里，就会奇迹般地变回一条完美的直线！

核心思想：

非线性回归的本质，就是寻找合适的函数变换规则，将未知的、复杂的非线性关系，转化为我们熟悉的线性回归模型进行求解。

【例 3-3】 双曲线模型代换与曲线拟合

代换过程

某市商店销售额 $x$ 与流通费率 $y$ 呈反比递减。选用双曲线方程：

\[ \hat{y} = a + b \frac{1}{x} \]

线性代换： 令 $x' = \frac{1}{x}$，则转化为线性模型 $\hat{y} = a + b x'$

代入数据计算得 $a=2.225, b=7.621$。
最终非线性模型为： $\hat{y} = 2.225 + 7.621 \left(\frac{1}{x}\right)$

选择指数模型

当增长呈“越多越快”的趋势时，可以考虑指数函数：

\[ y=ae^{bt} \]

对两边取自然对数：

\[ \ln y=\ln a+bt \]

令 $Y'=\ln y$，$A=\ln a$，就把弯曲的指数模型转化成线性模型 $Y'=A+bt$。

二、常见的非线性函数图形及其代换 (1/3)

通过合适的变量替换 $x', y'$，可以把曲线转化为直线 $y'=a'+bx'$ 求解：

1. 幂函数

原式: $y = a x^b$
代换: $y' = \lg y, x' = \lg x$

2. 指数函数

原式: $y = a e^{bx}$
代换: $y' = \ln y, x' = x$

二、常见的非线性函数图形及其代换 (2/3)

3. 对数函数

原式: $y = a + b \lg x$
代换: $y'=y, x' = \lg x$

4. S形函数 (Logistic曲线等)

原式: $y = \frac{1}{a + b e^{-x}}$
代换: $y' = \frac{1}{y}, x' = e^{-x}$

第四节 应用回归预测法应注意的问题

避免盲目套用模型，警惕伪回归

一、定性分析必须先行：警惕“伪回归”

经典谬误：冰激凌与溺水

某统计学家收集了某城市过去10年的数据，发现：“冰激凌的月销量”与“该月溺水死亡人数”呈现极高的正相关（$R^2 = 0.85$）。

如果仅仅迷信数据和模型，盲目得出“吃冰激凌会导致溺水”的结论，并建议政府“为了拯救生命应当禁止销售冰激凌”，这显然是荒谬的。

统计学本质揭秘：

这就是典型的“伪回归 (Spurious Regression)”。实际上，两者是由一个共同的隐蔽变量（Confounding Variable）——“夏季气温升高”驱动的。
结论：回归分析绝不能替代事物内部逻辑的定性分析。数学检验是第二位的，业务逻辑检验永远是第一位的！

二、不能任意外推：真理向前一步就是谬误

案例：化肥与产量的“边界”

某农业科研小组通过实验得出了“施肥量(x)”与“农作物产量(y)”的正相关线性回归方程：$\hat{y} = a + bx$。

在低浓度施肥区域 $[x_{\min}, x_{\max}]$，这个预测极其精准。但如果盲目认为只要无限增加施肥量就能获得无限产量（任意外推），预测值将脱离实际。
实际上，一旦超过合理阈值，过量施肥会导致“烧苗”，产量锐减，真实规律变为“倒U型”。

统计学法则：

回归模型只在原有数据的**插值范围**内可靠。对于远离样本均值的**外推预测 (Extrapolation)**，一方面置信区间呈喇叭口发散，误差极大；另一方面，业务逻辑的潜在结构可能早已发生扭曲。

三、数据资料的要求：警惕环境与结构的突变

案例：教培行业的“双减”断层

某教培企业打算预测 2022 年及之后的业务营收。如果分析师将 2018-2020 年的强劲增长数据，与 2021 年国家出台“双减”政策后的数据**简单混合在一起做一条回归直线**，得到的预测将毫无价值。

这是因为底层的“宏观政策变量”发生了**结构性断裂 (Structural Break)**，预测所依赖的“社会经济现象基本稳定”的假定被打破。

数据处理对策：

对于发生过重大技术革命、政策突变前后的数据，**不可将错就错直接合并**。必须应用分段回归（Piecewise Regression），或引入虚拟变量（Dummy Variable），或直接废弃突变前的失效历史数据。

本章小结

  • 一元线性回归： 最基础的模型。掌握最小二乘法估计参数 $b_0, b_1$，掌握标准误差、可决系数 $R^2$、t检验、F检验以及 D-W自相关检验。
  • 多元线性回归： 引入更多自变量。需注意多重共线性的检查（相关矩阵），并使用修正的可决系数 $R_a^2$ 评估模型优劣。
  • 非线性回归： 通过恰当的变量代换（取对数、倒数等），将曲线转化为直线，从而利用线性模型的方法求解参数。
  • 应用准则： 必须在定性分析确认因果关系的指导下进行（防伪回归）；警惕数据外推带来的误差（避免任意外推）；确保宏观环境的稳定性（防结构突变）。
